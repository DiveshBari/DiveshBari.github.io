PROJECT // 01

CarIT Microservices
Migration

Spring Boot · Spring Cloud · React

Took the legacy monolithic CarIT workshop module and broke it into independently deployable microservices — each owning a clean domain boundary and talking through a single API Gateway.

  • Four services: Customer, Vehicle, Planning, Order
  • Eureka for service discovery, OpenFeign for service-to-service calls
